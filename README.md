# LLVM_Modify简介

鉴于OLLVM在圈子内垄断的现象, `LLVM_Modify`应运而生, 它是一个免费的成品OLLVM, 适用于`AArch(32/64)`和`X86_64`平台, 即Linux/Windows客户端可使用的OLLVM。

特色：
- 支持`AArch64`, `X64`平台
- 优化`AArch64`平台的性能
- 安全性提高
- 对指令集进行改进, 支持控制流, 扁平化, 字符串隐藏
